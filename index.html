Projects

1. oneStopShop- an eCommerce app

This e-commerce application is built using React.js for the frontend, styled with Tailwind CSS, and powered by Redux Toolkit for efficient state management. The backend is handled by Appwrite, ensuring secure data storage and seamless user authentication.

Features

  1. User authentication (login/signup).
  2. Add to cart and wishlist functionalities.
  3. Syncs local storage with the database upon login.
  4. Persistent data storage across reloads.
  5. Fully responsive design for all devices.

2. Quiz.js- a javascript quiz app

Quiz.js is a dynamic and engaging quiz app designed to make learning fun and interactive. With a sleek interface and smart features, it ensures an immersive experience for users of all ages.

Features

  1. Data Retention: Uses local storage to save progress and results.
  2. Performance Feedback: Get detailed feedback at the end of the quiz to track your improvement.
  3. Timed Challenges: Countdown timer for each question adds excitement and focus.
  4. Flexible Navigation: Skip button lets you move past tough questions.
  5. Immersive Interactions: Audio toggle button for a customizable experience.
